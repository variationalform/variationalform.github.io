Getting Ready

The session will rely on a Jupyter notebook launched in a base Anaconda3 environment. To best prepare for the session you can

  • Create a folder called PythonForMaths somewhere convenenient (e.g. on your Desktop).
  • Create a subfolder called gfx.
  • Download the jupyter notebook to PythonForMaths.
  • Download either or both of the two photos to gfx.
  • Put your own jpeg photo in gfx if you want (optional).
If you are not going to use the Anaconda3 environment on the lab machines and instead use your own set up then these are the minimum requirements:
  • A python 3.x installation with some form of code editor.
  • A virtual environment as a sandbox for the event.
  • Jupyter notebook (strongly, very strongly, recommended).
  • scikit-learn, numpy, scipy, pandas, matplotlib, seaborn[stats] , ipython[all], PIL"
